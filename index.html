The System

Two-Stage Classification Pipeline

01 / INPUT

Image Acquisition

A clinical photograph is uploaded and preprocessed — resized to 224×224, normalised to [0,1], and passed as a tensor to the first model.

02 / STAGE 1

Category Classification

The first model performs binary classification, routing the image to either the allergy or infection specialist model for stage two.

Skin Allergy Skin Infection

03 / STAGE 2

Disease Classification

A specialised model identifies the exact condition — four allergy types or two infection types — returning a label and confidence score.
